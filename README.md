# zentflow
Zentflow is a 100% offline, peer-to-peer file sharing app built with Flutter, designed for ultra-fast transfers over local WiFi.
🚀 Zentflow

Zentflow is a blazing-fast, fully offline file sharing app built with Flutter. It enables seamless device-to-device transfers over local WiFi with a modern, glassmorphic UI and zero internet dependency.

---

✨ Features

⚡ High-Speed File Transfer

- Optimized for local WiFi (LAN) speeds
- Adaptive transfer system for maximum performance
- WARP Mode for multi-threaded, ultra-fast transfers

---

📡 Automatic Device Discovery

- Instantly detects nearby devices on the same network
- No pairing, PIN, or manual setup required
- Connect and start sharing in seconds

---

🔁 Bidirectional Sharing

- Both devices can send and receive files simultaneously
- No “host” or “receiver-only” limitation

---

📦 Multi-file Transfer & Queue

- Send multiple files at once
- Add files during ongoing transfers
- Smart queue system with sequential or parallel handling

---

⏸️ Full Transfer Control

- Pause, Resume, and Cancel transfers anytime
- Reliable resume from last progress point
- Real-time speed and progress tracking

---

📋 Clipboard Sharing

- Instantly share text between connected devices
- Copy received text with a single tap
- Displays sender device name

---

🎨 Modern UI (Glassmorphism)

- Clean and premium design
- Smooth animations and intuitive layout
- Consistent experience across platforms

---

🌗 Multiple Themes

- Dark Mode
- AMOLED Mode
- Light Mode (optimized for visibility)

---

📊 Transfer Statistics

- Track files transferred
- Monitor total data shared
- View performance insights

---

🔐 Fully Offline & Private

- Works entirely on local WiFi
- No internet required
- No external servers or data tracking

---

🐧 Cross Platform

- Android support
- Linux desktop support
- Consistent UI and performance

---

⚙️ Tech Stack

- Flutter — Cross-platform UI
- TCP Sockets — High-speed file transfer
- Local Network Discovery (mDNS / UDP)

---

🎯 Vision

Zentflow aims to deliver a fast, private, and beautifully designed alternative to traditional file sharing apps — combining performance, simplicity, and modern UI.
